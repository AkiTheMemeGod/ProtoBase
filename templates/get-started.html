Get Started with ProtoBase

Welcome to ProtoBase! Follow this guide to integrate and use our powerful backend platform for your application. Explore the API endpoints, check examples, and learn how to implement them seamlessly.

Quick Start

  1. Sign up for ProtoBase to get your API key.
  2. Use our RESTful API to interact with your backend.
  3. Leverage pre-built authentication methods and data management tools.

ProtoBase supports both REST and GraphQL endpoints for maximum flexibility. Jump to examples to see the API in action.

API Endpoints

Endpoint Method Description
/auth_api/email-signup GET Sign up with email, username, and password.
/auth_api/user-signup GET Sign up with username and password.
/auth_api/email-signin GET Log in using email, username, and password.
/auth_api/user-signin GET Log in using username and password.

Examples

Email Signup


GET /auth_api/email-signup?usr=johndoe&pwd=securepassword123&email=johndoe@example.com
Response:
{
    "message": "User Successfully SignedUp",
    "status-code": 200
}

Email Signin


GET /auth_api/email-signin?usr=johndoe&pwd=securepassword123&email=johndoe@example.com
Response:
{
    "message": "User Successfully Logged in",
    "status-code": 200
}

Username Signup


GET /auth_api/user-signup?usr=johndoe&pwd=securepassword123
Response:
{
    "message": "User Successfully SignedUp",
    "status-code": 200
}
